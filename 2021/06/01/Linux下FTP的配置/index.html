vsftpd: refusing to run with writable root inside chroot() 错误的解决办法
一下是解决办法：

为了避免一个安全漏洞，从 vsftpd 2.3.5 开始，chroot 目录必须不可写。使用命令：

chmod a-w /home/user

user 为FTP所连接的目录
