mysql发展史 一个男人的2个女儿名字命令数据库 1995年

MySQL是开源 关系数据库管理系统（RDBMS）。它的名字是“My”（联合创始人Michael Widenius的女儿的名字）和“ SQL ”（结构化查询语言的缩写）的组合。一个关系型数据库将数据组织成一个或多个数据表，其中数据类型可能相互关联；这些关系有助于构建数据。SQL 是程序员用来从关系数据库中创建、修改和提取数据以及控制用户对数据库的访问的语言。除了关系数据库和 SQL 之外，像 MySQL 这样的 RDBMS 与操作系统一起工作，以在计算机的存储系统中实现关系数据库，管理用户，允许网络访问并促进测试数据库完整性和创建备份。

MySQL 的第一个版本出现于 1995 年 5 月 23 日，它最初是从基于低级语言ISAM 的mSQL创建供个人使用的，创建者认为它太慢了和不灵活。他们创建了一个新的SQL接口，同时保持相同的API作为 mSQL。通过使 API 与 mSQL 系统保持一致，许多开发人员能够使用 MySQL 而不是（专有许可的）mSQL 前身。

MySQL 是GNU 通用公共许可证条款下的免费开源软件，也可在各种专有许可证下使用。MySQL 由瑞典公司MySQL AB拥有和赞助，该公司被Sun Microsystems（现为Oracle Corporation）收购。 2010年，当甲骨文收购Sun时，Widenius分叉了开源MySQL项目，创建了MariaDB。

MySQL 具有独立客户端，允许用户使用 SQL 直接与 MySQL 数据库交互，但更多情况下，MySQL 与其他程序一起使用以实现需要关系数据库功能的应用程序。MySQL 是LAMP Web 应用程序 软件堆栈（和其他）的一个组件，它是Linux、Apache、 MySQL 、Perl / PHP / Python的首字母缩写词。许多数据库驱动的 Web 应用程序都使用 MySQL，包括Drupal、Joomla、phpBB和WordPress。MySQL也被许多流行的网站使用，包括Facebook、 Flickr、 MediaWiki、 Twitter、和 YouTube。

MariaDB的是一个社区开发的商业化支持的叉的的MySQL的 关系数据库管理系统（RDBMS），意在保持免费和开放源码软件的下GNU通用公共许可证。开发由 MySQL 的一些原始开发人员领导，由于担心它在 2009 年被甲骨文公司收购，他们分叉了它。

MariaDB 旨在保持与 MySQL 的高度兼容性，确保具有库二进制奇偶校验和与 MySQL API和命令的精确匹配的直接替换能力。但是，新功能的差异更大。它包括新的存储引擎，如Aria、ColumnStore和MyRocks。

它的首席开发人员/CTO 是Michael “Monty” Widenius，他是MySQL AB的创始人之一，也是 Monty Program AB 的创始人。2008 年 1 月 16 日，MySQL AB 宣布已同意被Sun Microsystems以约 10 亿美元的价格收购。收购于 2008 年 2 月 26 日完成。第二年 Sun 被甲骨文公司收购。MariaDB 以 Widenius 的小女儿 Maria 的名字命名。（MySQL 以他的另一个女儿 My 的名字命名。）
