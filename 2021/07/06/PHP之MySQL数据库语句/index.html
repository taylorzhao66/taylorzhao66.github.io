MySQL数据库语句

数据库增删改查
SELECT column FROM table
WHERE column operator value

DELETE FROM table_name
WHERE column_name = some_value

UPDATE table_name
SET column_name= new_value
WHERE column_name = some_value

INSERT INTO table_name(column1,column2,…) VALUES (valuel, value2,…)

1.mysql_connect 打开一个到 MYSQL服务器的连接

$conn = mysql_connect("localhost:3306","root","root");

2.mysql_select_db 选择MySQL数据库

mysql_select_db("phpdemo",$conn);

3.mysql_query 查询数据库返回一个result的标识

$result = mysql_query("select * from user");

mysq1_ query（)仅对 SELECT, SHOW, DESCRIBE,EXPLAIN和其他语句语句返回resource
如果查询岀现、错误则返回 FALSE.

对于其它类型的SQL语句，比如 IINSERT, UPDATE,DELETE,DROP之类，mysq1 auer（）在执行成
功时返回TRUE,出错时返回 FALSE.

4. mysql_fetch 把result中的数据提取出来

mysql_fetch_assoc

$result_arr = mysql_fetch_assoc($result); 把result中的数据提取出来,一次只提取一条记录

mysql_fetch_assoc
从结果集中取得一行作为关联数组

mysql_fetch_row
从结果集中取得一行作为枚举数组

mysql_fetch_array
从结果集中取得一行作为关联数组，或数字数组，或二者兼有
